5. LABELING THE TRAINING DATASET

5. LABELING THE TRAINING DATASET

This chapter uses DeepLabCut's GUI. Leave the Anaconda Prompt as it is

5.1 Open

ClickLoad frames, then list of labeled-data folder will be shown. Choose a folder (not a file) whose name is the same with the video to be analyzed, and then click Open at the bottom right. Then, a new window pops up for labeling points.

5.2 Labeling

For every frame using the labeling window,

  • right click to create a new marker, left click to move the marker'.

  • Individual bodyparts need a maker. Once a marker is created for one bodypart, you need to move on to the next bodypart. The marker you are creating or moving can be selected on the Select a bodypart to label.

  • If some bodyparts cannot be found in a specific frame, skip that frame for the bodyparts.

  • A marker can be removed by left-clicking it and then pressing the delete key

  • Once you labels markers for all bodyparts, move on to the next frame by clickng Next.

Once you complete all frames, do not forget to click Save. Then, the window can be closed by clicking x. Quit will move on to a next video if any.

explanation of each button

  • Next and Previous change the frame to be shown.

  • Zoom
    By pressing Zoom, you can select a square region to be zoomed in. Home button will make the zoom back to 1.

  • Pan
    Pan moves a field ov view.

  • Home
    Make the view back to the default (after view is changed by Zoom or Pan buttons)

  • Lock Even when you move to a different frame, a field of view is locked.
