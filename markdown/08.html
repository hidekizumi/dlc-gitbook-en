8. IMPROVEMENT OF THE TRAINING RESULTS

This chapter uses Google Drive, Google Colabo, Anaconda Prompt

To improve the results, you need to select specific frames which perform poorly, and then re-label the training set to re-train the netwrok.

8.1 Download a project folder from the Google Drive

Download a project folder which Google Calb just made. Then put the folder in your local directory that you will run Anaconda Prompt (wich cd command)

8.2 Run Anaconda Prompt as an administrator, and create a virtual environment.

Do the same steps with 1. SETTING UP A VIRTUAL ENVIRONMENT and * 2. CREATION OF A PROJECT 2.2

8.3 Select frames which are expected to perform poorly

First, select frames that result in poor performance, using function deeplabcut.extract_outlier_frames(). There are four ways to judge which frames makes the performnace poor. The four approaches can be chosen as theoutlieralgorithmargument of functiondeeplabcut.extract_outlier_frames(). The default approach isoutlieralgorithm = 'jump'`

The first is to choose frames based on their low likelihood for each labeled bodypart. outlieralgorithm = 'uncertain'

  • the threshold for the likelifood can be set as the argument p_bound. The default value is p_bound = 0.01.

The second is based the acceptable range which moving-average fitting can predict in the xy coordinate for each labeled bodypart. outlieralgorithm = 'Fitting'

  • the accepatable range can be set as the argument epsilon. The default is epsilon = 20.

The third is based on the transient jump in the xy coordinate for each labeled bodypart. outlieralgorithm = 'jump'

  • the extent of jump is set as the argument epsilon. The defualt is epsilon = 20.

The last is maunal. `outlieralgorithm = 'manual'

8.4 Change the position of labelled markers.

function deeplabcut.refine_labels() readjust the position of labelled markers for exracted frames. 

deeplabcut.refine_labels(path_config)

A GUI will be started. This time, select a h5 file instead of a folder (different from deeplabcut.label_frames).

go to a folder task-researcher-yyyy-mm-dd, select machinelabels-iter0.h5, and then click Open

You will be asked about a threshold for the likelihood. Try 0.40 (defualt).

Then, for each extracted frames (with low likeliehoo), you can confirm the marker position. In some frames, you need to remove the makers, by pressing a mouse wheel (different from deeplabcut.label_frames). Complete all frames.

8.5 Mearge the newly corrected data as a trained set.

Function deeplabcut.merge_datasets() can merge the corrected data.


deeplabcut.merge_datasets(path_config)

Now the new training is ready to be run.
