7. Google Colaboratory

This chapter uses Google Drive and Google Colab

7.1 Upload the folder just created to the Google Drive directory

The folder 'task-researcher-yyyy-mm-dd' needs to be uploaded to Google Drive directory (e.g. My Drive > DeepLabCut >).

7.2 Uplaod a Notebook to the Google Drive directory

Download notebook from here and upload it to the Google Drive directory of task-researcher-yyyy-mm-dd This Google Drive directory needs to have a config.yaml.

Downloading can be done as File -> Download -> as .ipynb

7.3 Upload video files

In the Google Drive direcotory 'task-researcher-yyyy-mm-dd' has a folder 'videos'. Inside the folder, there is a avi file, but this is just a link file not an actual one. Replace the link file with an actual video file. Uploading the actual file to Google Drive can take time, but you can proceed without waiting for the uploading to be completed.

7.4 Open the Notebook with Google Colabo

The notebook just uploaded needs to be opend with Google Colabo. Right click and select Open with -> Google Colaboratory

From here, we will run codes in the notebook, so the relevant explanations are written there.
