6. RE-LABELING AND CONFIRMATION OF LABELLED FRAMES

This chaper describes steps with Anaconda Prompt. The steps in this chapter can be skipped

6.0 Re-label markers

New! : After automatic labeling, if you find some markers seldom labeled, you can select frames manually as described below. 

deeplabcut.extract_frames(path_config, 'manual')

How to select frames manually : Selection is based on a GUI, which is similar to labeling steps.

  • First, click Load Video, then select a video file insdie the 'videos' folder.
  • Next, you are asked "Do you Want to Crop the frames?". In most cases, "No" should be fine, but if you select "Yes", define a square with your cursor, and click Set.
  • Then, select frames by pressing Grab Frames. It is sometimes not obvious but one click is sufficient to select the frame.
  • If you move on to the next frame one by one, press → ←. If you use a scroll bar, you can jump to specific frames.
  • 20-30 frames are sufficient for one bodypart.
  • Once you finish the selection, close the window.

Once the frames are selected, mark labels as before. Here you need to go through the newly selected frames together with previously used ones. These frames are in time sequence, so unlabeled and already labeled frames are mixed. So all the frames need to be checked. 

deeplabcut.label_frames(path_config)

6.1 Create frames to confirm labeling

The following code creates the frames for confirmation. path_config is the same with one used in function label_frames().

deeplabcut.check_labels(path_config)

CAUTION: occassionally this funciton cannot be run. You can proceed without the confirmation of frames.

If you are unsure which directory you are now at, run the following code. 

import os
os.getcwd()

6.2 Inspect

The above code creates frames in the folder, labeled-data, insdie the folder, 'task-researcher-yyyy-mm-dd'.
