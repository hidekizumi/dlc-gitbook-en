1. SETTING UP A VIRTUAL ENVIRONMENT

1.1 Launch 'Anaconda Prompt' (command prompt) as an administrator

'Anaconda Prompt' can be found Start -> Anaconda3 (64-bit) -> Annaconda Prompt

Right-click 'Annaconda Prompt' and select 'More'-> 'Run as administrator'.

The administrator access is critical for DeepLabCut to output files

1.2 Set up a virtual environment

Run the following line in the Annaconda Prompt

conda activate deeplabcut

1.3 change a working directory

The default current directory should be C:\Windows\system32, but making new files there would be a mess. Therefore, it is highly recommended to prepare a separate direcotry, and change (cd command) to there as a working directory.

For expamle, if you want to change to a folder named DeepLabCut in the Documents folder of {UserName}'s, run the following line.

cd C:\Users\UserName\Documents\DeepLabCut

At the moment, the directory in the newtork drive S cannot be selected for unknown reasons.

This gitbook will describe the following steps, assuming you have a virtual environment at

  • C drive
    • Users
      • UserName
        • Documents
          • DeepLabCut

1.4 Copy video files to the working directory

The copying is not mandatory, but makes the following steps easy to process. This gitbook assumes that the video files are placed in the working directory.
